# Student UI Kit — Kapsul

The signature student-facing experience: sidebar + Hub + the **split-screen Chat AI** (chat left, editor right).

## Files
- `index.html` — interactive prototype starting at the Hub, click into Chat to see the split-screen
- `Sidebar.jsx` — fixed 240px sidebar, sections: Apprentissage, Studio Créatif, Intégrations
- `Hub.jsx` — greeting card, calendar/activity columns, to-do
- `Chat.jsx` — split-screen container
- `ChatPanel.jsx` — messages, mode toggle, composer with slash menu
- `EditorPanel.jsx` — Kapsul/Word/Notion tabs, doc surface
- `primitives.jsx` — Button, Pill, Card, IconBtn, KMark
